## 摘要
同策略蒸馏通过教师提供的密集、token级监督，提升了标准强化学习的学习效率。在OPD的标准KL目标中，token级损失被均匀平均，意味着所有token权重相等。然而，我们发现并非所有token都是平等的：随着学生轨迹变长，它们逐渐偏离教师的分布，导致后续位置的监督质量下降。因此，仅使用前30% token的OPD性能可与使用全部token相当，而仅使用后30% token的OPD几乎学不到任何东西。本文通过约束优化的视角，对这一问题的原理提供了理解。基于这些见解，我们推导出重要性加权同策略蒸馏，其中每个token的权重取决于学生与教师分布之间的累积差异，自然地对较早的token赋予更高权重，而对偏差较大的较晚token赋予较低权重。我们证明，IW-OPD的收敛速度显著快于OPD，学习效率更高，并且在相同规模和跨规模设置下均能达到比标准OPD更好的最终性能，在AIME-2025上性能提升高达6.9个点。

## Abstract
On-Policy Distillation (OPD) improves the learning efficiency of standard reinforcement learning through dense, token-level supervision from teachers. In the standard KL objective of OPD, token-level losses are uniformly averaged, implying equal weights for all tokens. However, we discover that not all tokens are created equal: as student rollouts grow longer, they deviate further from the teacher's distribution, leading to degraded supervision quality at later positions. As a result, OPD using only the first 30% of tokens can perform comparably to using all tokens, whereas OPD using only the last 30% of tokens barely learns anything. In this work, we provide a principled understanding of this issue through the lens of constrained optimization. Based on these insights, we derive Importance-Weighted On-Policy Distillation (IW-OPD), in which the weight assigned to each token depends on the accumulated discrepancy between the student's and teacher's distributions, naturally upweighting earlier tokens and downweighting later ones with larger deviations. We show that IW-OPD converges significantly faster than OPD, with better learning efficiency, and achieves better final performance than standard OPD in both same-size and cross-scale settings, improving performance up to 6.9 points on AIME-2025.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：在线策略蒸馏（OPD）通过教师模型提供密集的 token 级监督来提升学生模型的学习效率，但其标准 KL 目标对所有 token 均匀加权。本文发现，并非所有 token 的监督质量相等：随着学生自身生成的轨迹变长，学生分布逐渐偏离教师分布，导致后续 token 的监督质量下降。实验表明，仅使用前 30% token 的 OPD 性能可媲美使用全部 token 的 OPD，而仅使用后 30% token 的 OPD 几乎学不到任何东西。这一现象被称为“位置偏差”。
- **研究背景**：OPD 在 LLM 后训练中广泛应用，其梯度信号来源于 token 级别的逆 KL 散度。传统 OPD 忽略了学生采样带来的分布偏移问题，即采样自弱学生的轨迹会包含教师分布外的内容，使得后续 token 的教师监督不可靠。
- **整体目标**：理解位置偏差的成因，并设计一种新的加权方法，根据学生与教师的累积分布差异动态调整每个 token 的权重，从而提升 OPD 的学习效率和最终性能。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
- 将 OPD 的训练过程视为一个**有限预算的约束优化问题**：学生模型在每个局部更新中只能移动有限的距离（KL 预算），因此应优先将梯度预算分配给与教师分布兼容的早期前缀 token，而对后期偏离较大的 token 降低权重。
- 推导出一个**最优策略** \( q^*_\theta(y) = \frac{\pi_\theta(y) r_\theta(y)^\alpha}{Z_\alpha(\theta)} \)，其中 \( r_\theta(y) = \pi_T(y)/\pi_\theta(y) \) 是似然比，\(\alpha \in (0,1)\) 由预算 \(\rho\) 决定。该最优策略天然地给高似然比（即教师兼容前缀）的样本更高概率。
- 基于此提出**重要性加权 OPD（IW-OPD）**，在采样自学生的轨迹上，对每个 token 的 OPD 优势乘以一个归一化的前缀重要性权重。

### 关键技术细节
1. **重要性加权形式**：将原目标重写为在 \( \pi_\theta \) 采样下的加权期望：
   \[
   J^*_{\text{IW}}(\theta) = -\mathbb{E}_{y \sim \pi_\theta} \left[ \sum_{t=1}^T \text{sg}[\tilde{r}_t] \log \frac{\pi_\theta(y_t|y_{<t})}{\pi_T(y_t|y_{<t})} \right]
   \]
   其中 \(\tilde{r}_t = r_t^\alpha / Z_{\alpha,t}\)，\(\text{sg}[\cdot]\) 表示停止梯度，确保权重仅作为系数不参与梯度计算。
2. **梯度形式**：等价于将 OPD 的优势 \( A^{\text{OPD}}_t \) 乘以权重 \(\tilde{r}_t\)，形成 IW-OPD 优势：
   \[
   A^{\text{IW-OPD}}_t = -\text{sg}[\tilde{r}_t] (\log \pi_\theta(y_t|y_{<t}) - \log \pi_T(y_t|y_{<t}))
   \]
3. **稳定化技巧**：
   - **对数尺度缩放**：用对数似然比累加代替直接概率比，避免指数增长。
   - **无符号累积**：使用 \( |A^{\text{OPD}}_k| \) 而非有符号优势累积，消除正负抵消。
   - **标准化**：将重要性权重缩放到 [0,1] 区间。
   - **与 OPD 的插值**：最终权重 \( \tilde{r}^{\text{IW-OPD}}_t = 1 + \gamma (1 - \frac{\sum_{k<t} |A^{\text{OPD}}_k|}{\sum_{k<T} |A^{\text{OPD}}_k|})\)，其中 \(\gamma=0.5\) 为默认值，保留标准 OPD 的密集监督作为基底，额外分配预算给兼容前缀。

### 算法流程（简化）
- 从学生策略 \( \pi_0 \) 采样一条完整响应。
- 计算每个 token 的教师/学生 log-prob 以及 OPD 优势。
- 基于 token 级 OPD 优势的累积绝对值计算前缀权重。
- 在 PPO 框架中，使用 IW-OPD 优势进行内循环更新（clip 裁剪）。
- 每轮迭代后刷新采样策略。

## 6. 论文的主要结论与发现

1. **位置偏差存在且显著**：OPD 学习获益主要来自前缀 token，后部 token 的监督价值很低。
2. **理论解释**：通过有限预算约束优化导出最优策略应重加权似然比，解释为何早期 token 权重应大。
3. **IW-OPD 有效**：提出的重要性加权方法收敛更快、最终性能更高，在所有设置下一致超越 OPD。
4. **优势随教师-学生匹配度增大而增大**：更强的教师搭配更弱的学生时，IW-OPD 的相对增益最大。
5. **与现有方法正交**：可与 ExOPD 等奖励设计方法结合进一步提升性能。
6. **稳定化技巧必要**：直接使用原始似然比不稳定，需要无符号累积、标准化和 OPD 混合才能获得稳健提升。
