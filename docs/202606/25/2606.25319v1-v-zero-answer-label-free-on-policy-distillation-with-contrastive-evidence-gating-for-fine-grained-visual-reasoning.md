## 摘要
细粒度视觉推理要求多模态大语言模型（MLLMs）识别任务相关的视觉证据，并将其推理过程建立在局部图像区域上。现有的基于智能体的方法通常依赖具有可验证奖励的强化学习，或在大规模标注推理轨迹上进行监督微调，导致代价高昂的探索、人工设计的验证规则，或对文本监督的严重依赖。避免此类外部答案标签的一种自然方法是让学生自身采样的轨迹中学习，这指向了在线策略蒸馏（OPD）。为了理解OPD能为视觉推理提供什么以及不能提供什么，我们将其重新审视为无负样本的停止梯度对齐。这一视角表明，尽管OPD提供了有效的词元级修正，但其上限受限于缺少轨迹级判别能力。受这些观察启发，我们提出了V-Zero，一种基于对比证据门控的无答案标签视觉推理框架。V-Zero不使用任何标注的文本答案标签；相反，在训练过程中，它将一个与问题相关的区域裁剪与一个负视觉视图配对，以评估学生采样的轨迹并门控密集的词元级蒸馏。在多个视觉推理基准上的实验表明，V-Zero在保持强大泛化能力的同时，持续提升了细粒度视觉推理性能。值得注意的是，V-Zero比先前的监督微调方法快5倍以上，比强化学习基线快10倍以上。代码和数据集将在https://github.com/eVI-group-SCU/V-Zero发布。

## Abstract
Fine-grained visual reasoning requires multimodal large language models (MLLMs) to identify task-relevant visual evidence and ground their reasoning in local image regions. Existing agentic methods typically rely on reinforcement learning with verifiable rewards or supervised fine-tuning on large-scale annotated reasoning traces, leading to costly exploration, hand-designed verification rules, or heavy dependence on textual supervision. A natural way to avoid such external answer labels is to learn from trajectories sampled by the student itself, which points to On-Policy Distillation (OPD). To understand what OPD can and cannot provide for visual reasoning, we revisit it as negative-free stop-gradient alignment. This perspective shows that, although OPD provides effective token-level correction, its ceiling is constrained by the absence of trajectory-level discrimination. Motivated by these observations, we propose V-Zero, an answer-label-free framework for visual reasoning with contrastive evidence gating. V-Zero uses no annotated textual answer labels; instead, during training it pairs a question-relevant regional crop with a negative visual view to evaluate student-sampled trajectories and gate dense token-level distillation. Experiments on multiple visual reasoning benchmarks show that V-Zero consistently improves fine-grained visual reasoning while preserving strong generalization. Notably, V-Zero is more than 5$\times$ faster than previous supervised fine-tuning methods and more than 10$\times$ faster than reinforcement learning baselines. Code and dataset will be released at https://github.com/eVI-group-SCU/V-Zero

## 2. 论文提出的方法论

### 核心思想
- 将 OPD 重新解读为 **无负样本的停止梯度对齐**（negative-free stop-gradient alignment），指出其仅提供词元级修正，缺少对整体轨迹正确性的判别。
- 引入 **对比证据门控机制**：学生从全图采样多条轨迹，教师用与问题相关的正裁剪视图（正证据）和无关区域的负裁剪视图（负证据）回放每条轨迹，通过对比生成轨迹级门控权重，加权正视图上的 OPD 损失。

### 关键技术细节
- **学生侧**：给定全图提示 `x`，学生 `π_s` 采样 G 条轨迹 `Y(x) = {y^(g)}`。
- **教师侧**：对每条轨迹，教师 `π_t` 在三种条件下计算对数概率：
  - 正视图条件：`(x, z^+, y^(g)_{<k})`
  - 负视图条件：`(x, z^-, y^(g)_{<k})`
  - 其中 `z^+` 是问题相关区域裁剪，`z^-` 是原图 2× 下采样后随机采样的等大无关区域裁剪。
- **证据差距计算**：每个学生采样 token 的教师侧证据差距 `Δ^(g)_k = log π_t(y^(g)_k | x, z^+, y^(g)_{<k}) - log π_t(y^(g)_k | x, z^-, y^(g)_{<k})`。
- **轨迹级证据分数**：`p^(g) = (1/T_g) Σ_k Δ^(g)_k`。
- **归一化与门控**：对同一 prompt 的 sibling 轨迹进行标准化：`a^(g) = (p^(g) - μ_x)/σ_x`，然后 `w^(g) = clip(1 + a^(g), w_min, w_max)`（停止梯度）。
- **蒸馏目标**：仅保留正视图 OPD，用门控权重加权平均：
  ```
  L_V-Zero = (1/G) Σ_g w^(g) * (1/T_g) Σ_k D_KL^(g,k)_+
  ```
  实际使用采样 token 的逆 KL 代理损失。

### 流程概要
- 训练时：学生采样轨迹 → 教师计算正/负视图对数概率 → 计算门控权重 → 加权正视图 OPD → 更新学生。
- 推理时：仅使用学生模型全图输入，无需额外裁剪或工具调用。

## 6. 论文的主要结论与发现

1. **理论发现**：OPD 可视为无负样本停止梯度对齐，其瓶颈在于缺少轨迹级判别。
2. **方法有效性**：V-Zero 通过正负视觉证据对比产生的轨迹级门控，弥补了 OPD 的该缺陷。
3. **性能提升**：在 Qwen3.5-4B 基础上，平均提升 3.1 分，且不损害通用能力（MMStar 从 71.8 提升至 74.4）。
4. **效率优势**：训练无需答案标签，速度比 SFT 快 5×以上，比 RL 快 10×以上；推理时零额外开销。
5. **泛化性**：在分布外基准（MMStar）上也取得提升。

## 7. 优点

- **方法创新性**：将对比学习思想引入 OPD 的轨迹级评估，形式优雅，理论动机清晰。
- **实用性强**：无需标注答案、无需人工奖励函数、无需推理时工具调用，可轻松集成到现有 MLLM 训练流程。
- **训练高效**：仅需 23K 数据、4.8 小时在消费级 GPU 上即可完成，远快于 SFT/RL 方法。
- **消融全面**：从门控、师生尺寸、rollout 数、训练步数等多角度验证了设计选择。
- **跨系统对比有说服力**：尽管骨干不同，V-Zero 在多个基准上达到或超越专用推理系统。

## 8. 不足与局限

- **对训练数据裁剪的依赖**：需要问题相关裁剪（正视图）作为教师输入，这本身可能是一种隐形监督；负裁剪生成方式（2× 下采样后随机区域）是否最优未深入探索。
- **消融不够完整**：未分析不同门控函数（如 sigmoid）、不同负采样策略（如基于注意力）、教师大小与其他骨干的更多组合；未进行多次独立运行或多种子验证。
- **实验仅针对单尺度模型**：学生仅为 4B 规模（Qwen3.5-4B），未扩展到更大模型（如 7B/9B）或不同架构，泛化性有限。
- **分布外泛化测试单一**：仅使用了 MMStar，不足以证明广泛的 OOD 能力。
- **对比公平性**：跨系统对比时，基线方法（如 DeepEyes, ZwZ）基于不同骨干（Qwen2.5/Qwen3），但论文未进行相同骨干下的公平比较（如 Qwen3.5-4B vs 用 Qwen3.5-4B 训练的 ZwZ）。
- **失败案例分析缺失**：未讨论哪些样本类型 V-Zero 仍会失败，或何时对比证据差距失效。
- **门控机制潜在风险**：如果正负视图差距小导致门控平坦，门控效果可能等同于无门控，但论文未分析该情况下的鲁棒性。
